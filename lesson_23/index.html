Які з віріантів невірні?

Вaріант №1

console.log('35' + - "22");

Верне НЕ 13. Верне 35-22, тому що строки при додаванні (їх значення) будуть виводитися в рядок

Вaріант №2

console.log('35' * "22");

Верне 770. Відповідь вірна. Оператор множення переводить значення string в number

Вaріант №3

console.log('558' > 22++);

НЕ Верне true. Синтаксична помилка. Інкремент має бути примінений тілько до змінної

Вaріант №4

let usersCounter = 0;
let newUsers = usersCounter++;
console.log(newUsers);

НЕ Верне 1. Верне 0, бо інкремент з постфіксним записом

Вaріант №5

console.log(!false && 11 || 18 && !'');

Верне НЕ 18. Верне 11, бо це перший true

Вaріант №6

let name = 0;
console.log(name ?? "Без імені");

Верне 0, бо це перший аргумент, який не null і не undefined
